Ρυθμίσεις Εφαρμογής

Αυτή η σελίδα σας επιτρέπει να προσαρμόσετε την εφαρμογή σύμφωνα με τις ανάγκες σας. Μπορείτε να ενεργοποιήσετε τη σκοτεινή λειτουργία, να ρυθμίσετε τις ειδοποιήσεις, να επιλέξετε τη γλώσσα της εφαρμογής και να συγχρονίσετε τα δεδομένα σας.
Επίσης, έχετε τη δυνατότητα να επαναφέρετε τις ρυθμίσεις στις εργοστασιακές προεπιλογές αν το επιθυμείτε.
